1) What is the definition of server port?


The answer is ___________

A server port is a numerical label assigned to each end point of communication in a network. 
It is used to identify a specific process running on a server, and to direct incoming network 
traffic to the appropriate service or application. 

2) What is server port range?

The answer is ___________
A server port range refers to a range of consecutive port numbers that are assigned to a
 specific service or application on a server.

3) How many bits does number above represent?

A server port is a 16-bit unsigned integer. It ranges from 0 to 65535.
